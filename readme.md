# 🏠 청년주택 STO 플랫폼 프로젝트 (HUG x ASTO)

본 프로젝트는 `asto.txt`의 핵심 비즈니스 모델을 바탕으로, 청년 주거 문제 해결을 위한 혁신적인 금융 구조(STO)를 제안하는 **원페이지 스크롤 발표용 웹사이트** 제작 계획입니다.

## 🚀 프로젝트 개요
- **주제**: HUG(주택도시보증공사)와 ASTO의 협력을 통한 청년주택 공급 및 STO 모델
- **핵심 금융 구조**: **40(자기자본) : 60(토큰증권)**
- **ASTO 역할**: 토큰 발행 및 유동성 공급 플랫폼 운영

## 📅 개발 및 발표 계획 (Roadmap)

### 1단계: 콘텐츠 기획 및 구조 설계
- [ ] **인트로**: 청년 주거 문제의 현황과 임팩트 있는 문구
- [ ] **파트너십**: HUG x ASTO 협력 모델의 신뢰성 강조
- [ ] **핵심 모델**: 40:60 자본 구조 시각화 (D3.js 또는 CSS 애니메이션 활용)
- [ ] **기술력**: ASTO의 토큰 발행 기술 및 보안성 설명
- [ ] **기대효과**: 투자자 수익성 및 청년 주거 안정성

### 2단계: 디자인 및 퍼블리싱 (One-Page Scroll)
- **디자인 컨셉**: 고해상도 이미지, 다크 모드 기반의 고급스러운 프리젠테이션 느낌
- **스크롤 인터렉션**: 
  - 섹션별 풀스크롤(Full-page) 전환 효과
  - 요소들이 스크롤에 따라 나타나는 Fade-in/Slide-up 애니메이션
  - 40:60 비율을 직관적으로 보여주는 인터렉티브 차트

### 3단계: 기술 스택 선정
- **Frontend**: HTML5, Vanilla CSS (Flexbox/Grid), JavaScript (ES6+)
- **Animations**: GSAP (GreenSock) 또는 ScrollMagic (부드러운 스크롤 효과)
- **Visualization**: D3.js (40:60 도넛 차트 또는 트리맵 시각화)

## 🎨 주요 섹션 구성안 (HTML 구조)
1. **Section 1: Hero**: "청년의 꿈, 조각 투자로 집을 짓다" (배경 영상/이미지)
2. **Section 2: Problem & Solution**: 주거난 지표와 STO 솔루션 제시
3. **Section 3: Partnership**: HUG와 ASTO의 로고 및 협력 체계도
4. **Section 4: Financial Model (Core)**: 40% 자기자본 vs 60% 토큰증권 비중 시각화
5. **Section 5: ASTO Platform**: 토큰 발행 프로세스 (Step-by-step 애니메이션)
6. **Section 6: Call to Action**: 프로젝트 참여 및 기대 수익 안내
